# GitFeed

![Status](https://img.shields.io/badge/status-finished-success)
![Vanilla JS](https://img.shields.io/badge/JavaScript-Vanilla_JS-f7df1e)
![GitHub Pages](https://img.shields.io/badge/deployed-GitHub_Pages-blue)

A social media like GitHub repository discovery platform built with vanilla JavaScript.

GitFeed makes it easier to discover interesting open-source projects through a clean feed user-interface, topic filters, search, and GitHub integration.

## Features

- Browse popular GitHub repositories
- Search repositories using GitHub Search
- Filter repositories by topic
- Infinite scrolling feed
- GitHub authentication
- Star and unstar repositories directly from GitFeed
- Personalized starred repository tracking
- Trending repository indicators
- Language color detection
- Repository statistics including:
  - Stars
  - Forks
  - Recent activity
- Responsive GitHub-inspired UI

## How It Works

Repositories are fetched from the GitHub API and displayed in a feed format. Users can browse by topic, search for projects, and authenticate with GitHub to star repositories directly from the application.

Starred repositories are synchronized using Supabase.
